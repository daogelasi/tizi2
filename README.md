测试标准
关于翻墙，目前已经有许多网站提供了科学上网或是VPN推荐，就我所知很多评测网站本身就是由翻墙软件厂商制作，作为宣传方式之一。然而我们并无偏袒任何VPN、SSR 节点，我们的 vpn 排行随着时间而调整变动，所有推荐皆有数据做支持

翻墙成功率
我们的评测第一标准在于软件能否国内翻墙，不论是SSR节点、vpn软件，只要测试后不能成功翻墙的，均不会进一步进行评分与推荐，目前我们仅推荐「中国还能用的vpn」。

连接速度
连接速度指的是用户点击软件中连线，到实际连上且翻墙成功所需要的时间。某些VPN需要接近二十分钟才能连上，对于使用体验来说相当差劲，我们会将所有连接速度在0~10分进行评分，最快连上的为10分。

下载上传速度
翻墙后的速度表现重要性因人而异，主要取决于观看哪种墙外网站，有些人翻墙主要使用国外视频网站，例如访问youtube，这种用户对于速度的要求较高。我们将下载与上传速度依照0~10分的标准进行评分，速度最快为10分。

价格
在翻墙 vpn选择或ssr节点购买上，我们都希望能获得便宜的vpn、好用的梯子，甚至看看是否有VPN试用。所以在翻墙软件推荐中，我们也会将价格进行0~10分的评分，10分为性价比最高的翻墙vpn。

其他
我们「翻墙 软件」的评分标准中亦包含安全性，例如wireguard 相比shadowsocks 安全许多；产品的通用性，像是ios翻墙能成功，安卓翻墙是否也能成功，或是其他手机翻墙表现如何；延迟度，进行游戏时或是视讯会议会不会有延迟的情况发生等等。
